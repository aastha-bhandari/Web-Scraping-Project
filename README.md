# Web-Scraping-Project

# About
This is my web scraping project created in VS Code with Jupyter Notebook support using Python libraries like Pandas, Matplotlib, BeautifulSoup and Requests
Web Scraping is a technique to extract large amount of data from a website instead of copying them manually.
I have used bookstoscrape website and extracted its data for my project.

# What it Does
-collects and parse required HTML. <br>
-scrapes data from website. <br>
-extract useful information. <br>
-saves the data as csv file. <br>
-cleans and visualise the data. <br>

# Libraries Used
-Pandas : stores the data in tabular form and convert it into csv format. <br>
-Matplotlib : helps in creating graphs for better visualisation. <br>
-BeautifulSoup : parses required HTML and extract required data from website. <br>
-Requests : fetches the webpage content. <br>
